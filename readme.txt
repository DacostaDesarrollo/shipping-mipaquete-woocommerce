Mipaquete.com te permite seleccionar la transporta ideal

== Description ==

Integración con mi [mipaquete.com](https://mipaquete.com/) plataforma especializada en envíos para e-commerce (Actualmente solo para Colombia). Con este plugin puede:

- Cotizar envíos nacionales y urbanos con diferentes transportadoras como TCC, COORDINADORA, SAFERBO, entre otras.
- Seleccionar de forma automática priorizando por precio precio, tiempo de entrega o calificación de servicio de la transportadora o con reglas predeterminadas durante la confirguración (Reglas como por ejemplo, seleccionar una de las transportadoras dependiendo del destino).
- Incluye la opción de envíos con Pago Contra Entrega.
- Con la implementación de este plugin, el e-commerce automatiza sus actividades logísticas. Para su configuración reciba soporte gratuito de parte del equipo de [mipaquete.com](https://mipaquete.com/)


== Installation ==

1. Descargar el plugin
2. Ingrese a su administrador de wordpress
3. Ingrese al menu plugins / añadir nuevo
4. Busque por el nombre **Shipping Mipaquete Woocommerce**
5. Instale y active el plugin y siga el proceso de configuración.
6. Establezca los detalles del envío del producto, ver captura de pantalla.
